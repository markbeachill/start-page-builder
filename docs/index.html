How it works

A small workflow for a practical result. You can keep the page local, or publish it online and use it anywhere.

Choose a template
Start with UK everyday links, US everyday links, a workplace menu, a jump-start template, or a blank menu.
Customise sections and links
Use sections you want, add your own sites, rename sections, choose 2–5 desktop columns, pick Primary or Pastel colours, show or hide the title, and preview as you go.
Save your start page
Save the standalone start page and a menu config file if you want to edit it again later.
The exported start pages are standalone-friendly and work on desktop, tablet and mobile.
