Help and notes

The builder runs in your browser. It is designed to create a standalone HTML start page and a project JSON file you can re-import later.

Saving your work

Use Download project JSON to keep an editable copy. Browser auto-save is convenient, but it is not a backup.

Using the HTML

Use Download HTML to create the page you can open locally or set as a browser homepage/start page.

Privacy

Your links are edited locally in your browser. This static GitHub Pages site does not need an account or upload your menu to a server.

Import formats

The builder can import project JSON, friendly template JSON, Markdown, CSV, and a strict plain-text format.

Markdown example

# My Menu

## Search
- [Google](https://www.google.co.uk/)
- [Wikipedia](https://www.wikipedia.org/)

## News
- [BBC News](https://www.bbc.co.uk/news)

CSV example

section,label,url,colour
Search,Google,https://www.google.co.uk/,blue
News,BBC News,https://www.bbc.co.uk/news,red

Plain text example

[Search]
Google | https://www.google.co.uk/
Wikipedia | https://www.wikipedia.org/
The builder page uses a light top-nav wrapper so the workspace and exported menus remain standalone-friendly.
