Help and notes

The builder runs in your browser. It creates a standalone start page and an editable project file you can open again later.

Build with AIPublish with GitHub Pages

Saving your work

Use Save editable project to keep a copy you can reopen later. Browser auto-save is convenient, but it is not a backup.

Using the start page

Use Download start page to create the standalone page you can open locally, set as your browser homepage, or publish online.

Mobile support

The classic layout gives you five columns on desktop and stacks neatly on mobile, so the exported page is usable on phones and tablets.

Privacy

Your links are edited locally in your browser. This static GitHub Pages site does not need an account or upload your menu to a server.

Open existing work

Use Open saved project/template to import your editable project JSON, a friendly template JSON, or an older builder config.

Examples

Open the UK, US and developer examples on the templates page to see real exported start pages.

Import formats

The builder can import project JSON, friendly template JSON, Markdown, CSV, and a strict plain-text format.

Markdown example

# My Start Page

## Search
- [Google](https://www.google.co.uk/)
- [Wikipedia](https://www.wikipedia.org/)

## News
- [BBC News](https://www.bbc.co.uk/news)

CSV example

section,label,url,colour
Search,Google,https://www.google.co.uk/,blue
News,BBC News,https://www.bbc.co.uk/news,red

Plain text example

[Search]
Google | https://www.google.co.uk/
Wikipedia | https://www.wikipedia.org/
