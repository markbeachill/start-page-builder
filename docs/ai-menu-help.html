Ask an AI to build your menu.

You can ask an AI assistant to create a list of links for you, then import it into the builder. Markdown is the easiest format to read and check, but CSV and JSON also work.

Open builderImport help

Recommended: ask for Markdown

Markdown is readable, easy to edit, and imports well.

Prompt to copy

Create a personal start page menu for me in Markdown format.

Use this exact structure:
# My Start Page

## Section name
- [Link label](https://example.com/)
- [Another link](https://example.com/)

Please create 8 to 12 useful sections, with 4 to 8 links in each section. Use practical, well-known sites. Do not include explanations after the Markdown.

Example output

# My Start Page

## Search & AI
- [Google](https://www.google.co.uk/)
- [ChatGPT](https://chatgpt.com/)
- [Wikipedia](https://www.wikipedia.org/)

## News
- [BBC News](https://www.bbc.co.uk/news)
- [Google News](https://news.google.com/)

In the builder, open More options and use Import Markdown.

CSV format

Useful if you want to work in a spreadsheet first.

section,label,url,colour
Search,Google,https://www.google.co.uk/,blue
News,BBC News,https://www.bbc.co.uk/news,red

JSON format

Useful for advanced users or when asking an AI to create a ready-made template.

{
  "title": "My Start Page",
  "filename": "start-page.html",
  "sections": [
    {"name":"Search","colour":"blue","links":[{"label":"Google","url":"https://www.google.co.uk/"}]}
  ]
}

Plain text format

Useful for quick hand-written lists.

[Search]
Google | https://www.google.co.uk/
Wikipedia | https://www.wikipedia.org/
Tip: ask the AI for links you recognise and trust, then review the result before importing. You can delete or edit anything in the builder.
The exported start pages are standalone-friendly and work on desktop, tablet and mobile.
