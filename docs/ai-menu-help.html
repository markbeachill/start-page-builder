Ask · refine · import

Ask an AI to help design your start page.

The easiest route is to ask an AI assistant to interview you first, suggest sections, then create a Markdown list you can import into the builder.

Open builderImport help

Recommended workflow

Use AI for the thinking and first draft, then use the builder for final editing and saving.

1. Ask the AI to interview you
Country, work, study, services, interests, media, travel, shopping, public services and tools.
2. Review the proposed sections
Ask the AI to add, remove or rename sections before it creates the full list.
3. Ask for Markdown
Markdown is readable, easy to check, and imports well into the builder.
4. Import and edit
Use Import list in the builder, then save your start page and menu config.

Best prompt to copy

I want to create a personal start page of useful website links.

Please ask me a few questions first about:
- my country or region
- the services I use
- my work, study or daily routine
- hobbies and interests
- news, media, sport or entertainment preferences
- shopping, money, travel, health or public-service needs

After I answer, suggest a set of menu sections and wait for me to approve or edit them.
Then create the menu as Markdown using this exact format:

# My Start Page

## Section name
- [Website label](https://example.com/)
- [Website label](https://example.com/)

Use real website URLs. Keep each section manageable. Do not include explanations after the Markdown.

Quick profile prompt

Use this when you already know what you want.

Create a start-page menu in Markdown for a retired teacher in the UK who uses Gmail, BBC iPlayer, online banking, gardening sites, local history websites, train travel, NHS services and radio. Group the links into useful sections. Use real URLs.

Developer prompt

Useful for specialist menus.

Create a start-page menu in Markdown for a freelance software developer in the US who uses GitHub, AWS, Vercel, React, Python, Figma, Stripe, Google Workspace and online learning sites. Group links by daily workflow.

Family computer prompt

Good for shared household start pages.

Create a start-page menu for a family computer shared by adults and children. Include search, school, streaming, shopping, travel, health, public services and safety tools. Use UK websites where possible. Output Markdown only.

Import formats

Markdown is the best default for AI-generated menus. CSV and JSON are available if you need them.

Markdown format

# My Start Page

## Search & AI
- [Google](https://www.google.co.uk/)
- [ChatGPT](https://chatgpt.com/)

## News
- [BBC News](https://www.bbc.co.uk/news)

CSV format

section,label,url,colour
Search,Google,https://www.google.co.uk/,blue
News,BBC News,https://www.bbc.co.uk/news,red

Menu config JSON

More advanced. This is the builder-readable configuration format.

{
  "title": "My Start Page",
  "filename": "start-page.html",
  "sections": [
    {"name":"Search","colour":"blue","links":[{"label":"Google","url":"https://www.google.co.uk/"}]}
  ]
}
Tip: ask the AI for links you recognise and trust, then review the result before importing. You can delete or edit anything in the builder.
The exported start pages are standalone-friendly and work on desktop, tablet and mobile.
